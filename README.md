# 🚁 SkyForged: The Drone Systems Operating Layer

**SkyForged** is a professional-grade, multi-modular platform designed to build, analyze, and secure drone systems. It serves as the interactive core for the [Drone Integration Handbook](https://saitanveesh.github.io/skyforged/handbook.html), providing the mathematical and tactical foundation for the next generation of UAV operators and engineers.

---

## ⚡ The Operating Layer

SkyForged is designed as a "System OS" for drone builders. Instead of scrolling through static documentation, users navigate through specific **Mission Paths**:

*   **⚡ Mission 01: Trainee** → Start the [SkyForged Academy](https://saitanveesh.github.io/skyforged/academy.html). A 12-phase system-level induction.
*   **🏗️ Mission 02: Architect** → Enter the [Model Builder](https://saitanveesh.github.io/skyforged/builder.html). Assemble platforms from **3,800+ vetted parts** with real-time constraint validation.
*   **📡 Mission 03: Engineer** → Run [RF Tactical Math](https://saitanveesh.github.io/skyforged/tools.html). Fresnel zone calculation, terrain link analysis, and link budgeting.
*   **🎯 Mission 04: Tactician** → Deploy [Ops Center](https://saitanveesh.github.io/skyforged/ops.html). Real-world field calculators and standard operating procedures.

## 🏗️ The Architecture (Cyber-Physical Security)

SkyForged is authored by **Sai Tanveesh**, a Cybersecurity student specializing in the security of autonomous UAV systems. 

Unlike generic FPV guides, SkyForged focuses on the **system-level approach**—understanding how flight controllers, MAVLink protocols, and hardware components interact in potentially vulnerable environments. The mission is to empower pilots with the technical literacy to secure and optimize real-world autonomous platforms.

## 🚀 Technical Stack

*   **Logic:** Pure, high-performance JavaScript (No external frameworks / zero-latency).
*   **Styling:** Modern HUD-style UI with custom-built keyframe animations.
*   **Data:** A massive 3,840-item curated component database.
*   **Hosting:** Fully static, edge-localized deployment via GitHub Pages.
